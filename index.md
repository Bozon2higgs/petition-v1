# International Petition – Version 1.0

A multilingual, legally grounded reference document addressing the relationship between public debt, democratic consent, and fundamental rights.

## About this project

This project provides access to a formally structured petition text, supported by international legal sources and established doctrine. It is intended as a documentary and civic reference resource.

## What this project is
- An international reference document
- A legally grounded and documented text
- A multilingual publication
- A reusable civic resource

## What this project is not
- Not a legal complaint
- Not a political campaign
- Not a judicial action
- Not an advocacy organization

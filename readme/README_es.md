
# Petición internacional – Versión 1.0

## Qué es la petición
Petición internacional sobre los efectos de las deudas públicas ilegítimas en los derechos humanos fundamentales.
Documento de referencia, versión 1.0.

## Contenido del repositorio
Directorio `petition-texts/` con un archivo completo por idioma.

## Idiomas de referencia
- Francés (FR)
- Inglés (EN)
- Español (ES)

## Lo que NO es
No es una sentencia, ni una demanda, ni un llamado a la ilegalidad.
Es un recurso documental y preparatorio.


# عريضة دولية – الإصدار 1.0

## ماهية العريضة
عريضة دولية تتعلق بآثار الديون العامة غير المشروعة على حقوق الإنسان الأساسية.
وثيقة مرجعية مستقرة وقابلة للتتبع.

## محتوى المستودع
مجلد `petition-texts/` يحتوي على نص كامل لكل لغة.

## اللغات المرجعية
الفرنسية، الإنجليزية، الإسبانية.

## ما ليست عليه
ليست حكمًا قضائيًا ولا دعوى قانونية ولا دعوة إلى عمل غير قانوني.

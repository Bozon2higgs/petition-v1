
# Международная петиция – Версия 1.0

## О петиции
Международная петиция о последствиях нелегитимных государственных долгов для прав человека.
Справочный документ, версия 1.0.

## Содержание репозитория
Каталог `petition-texts/` с полным текстом для каждого языка.

## Справочные языки
Французский, английский, испанский.

## Что это НЕ
Не судебное решение и не иск.

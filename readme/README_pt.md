
# Petição internacional – Versão 1.0

## O que é a petição
Petição internacional sobre os efeitos das dívidas públicas ilegítimas nos direitos humanos fundamentais.
Documento de referência, versão 1.0.

## Conteúdo do repositório
Diretório `petition-texts/` com um arquivo completo por idioma.

## Idiomas de referência
Francês (FR), Inglês (EN), Espanhol (ES)

## O que NÃO é
Não é decisão judicial nem ação legal.

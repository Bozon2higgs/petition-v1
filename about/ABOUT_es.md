# Sobre este proyecto

## 1. Finalidad del proyecto

Este proyecto proporciona un marco de referencia internacional, multilingüe y neutral para la difusión y reutilización de un texto de petición de alcance global.

---

## 2. Metodología

El texto se desarrolló a partir de marcos jurídicos internacionales reconocidos, con coherencia lingüística entre versiones.

---

## 3. Neutralidad editorial

El proyecto no tiene afiliación política ni institucional.

---

## 4. Posicionamiento jurídico

No constituye una acción judicial ni una denuncia legal.

---

## 5. Gobernanza y versiones

Versión actual: **1.0**.

---

## 6. Uso y reutilización

El texto puede ser citado y reutilizado libremente con atribución.

---

## 7. Idiomas y accesibilidad

Todas las versiones buscan equivalencia de significado.
